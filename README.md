# Threadline Navigator 5.4a

React/Vite workspace for the Threadline Navigator family dashboard.

## Run Locally

1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env.local` and fill in the required API keys.
3. Start the dev server:
   `npm run dev`

## Useful Scripts

- `npm run dev` starts Vite on port 3001.
- `npm run build` creates the production build.
- `npm run lint` runs TypeScript with `--noEmit`.
